# 프로그래밍 개요

### 프로그램
  - -> 가공되지 않은 데이터를 가지고 알고리즘을 이용하여 원하는 결과를 도출하는 과정 
  > **Note:** 알고리즘 : 명령어들의 조함
------------------------------------
### S/W설계
  - 요구사항 분석
  - flow chart 설계 => 구현/설계
  - 코드 구현
  - 실행
  - 테스트
  - 완료
 
### Cross Compiler
  - 예) intel에서 컴파일 한 후 Arm용 실행파일(바이너리파일)을 만드는 것
  - compiler  : code -> .obj
  - linker    : .obj + .obj + .h + .lib + ...

### 기타 용어
  - 전처리기 : #~~~~, 컴파일 전 처리하는 명령어
  
### C언어 문법의 흐름
```sequence
변수(일반 변수) -> 배열(일반 배열) -> 함수 -> 함수포인터
                                     +
                                포인터(싱글) -> 더블 포인터
                                
                                  (배열화)   -> 포인터 배열
                                
                 배열(일반 배열) -> 2차원배열 -> 배열 포인터
```
